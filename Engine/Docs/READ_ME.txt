# Westlake Engine

Westlake Engine is an open-source game engine built in C++, designed to provide developers with a flexible and powerful framework for game development.

## Description

Westlake Engine is a custom game engine that aims to offer:
- High-performance rendering capabilities
- Modular architecture for easy extension
- Cross-platform compatibility
- Real-time physics simulation
- Modern graphics pipeline
- Intuitive scene management

## Getting Started

### Prerequisites

* C++17 or later
* CMake 3.15+
* Graphics card with support for modern graphics APIs
